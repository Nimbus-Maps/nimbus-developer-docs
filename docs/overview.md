# Nimbus Developer Docs

Nimbus developer documentation is organised by integration surface. Each API has its own getting started guide, quick reference, examples, and full OpenAPI reference.

## Available APIs

| API | Use it for | Guides | Reference |
|-----|------------|--------|-----------|
| Property Search API | Search UK property titles, comparable deals, addresses, planning data, constraints, EPCs, ownership, and related property intelligence. | [Property Search guides](property-search/overview) | [OpenAPI reference](/api/property-search/) |
| Document Purchase API | Check document availability, purchase HMLR and property-related documents, receive webhook delivery events, and download purchased files. | [Document Purchase guides](document-purchase/overview) | [OpenAPI reference](/api/document-purchase/) |
